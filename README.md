### Instalación 🔧

Para ejecutar el código se debe 

*Clonar el repositorio en la pc

```
git clone https://github.com/FrancoPisapia/React-Project.git
```

*Instalar el node modules del proyecto

```
npm i
```

*Una vez instalado ejecutar el comando

```
npm start
```
Y listo
